Волновое уравнение

Колебания разной природы описываются волновым уравнением: \[\nabla^2 \xi = \dfrac 1 {v^2}\pdv[2]{\xi}{t},\] где \(\xi = \xi(\bf r, t)\) — возмущение в точке \(\bf r\) в момент \(t\), \(v\) — фазовая скорость распространения волны.

  • Возмущение
  • Волновой пакет
  • Гармоническая волна
  • Затухающая в.
  • Стоячая в.
Код 
from matplotlib import pyplot as plt
from matplotlib.animation import FuncAnimation
import numpy as np
from IPython import display
from functools import partial

def plot(wave_func):
    fig = plt.figure()
    fig.set_size_inches(7, 2, True)
    ax = plt.axes(xlim=(0, 3 * np.pi), ylim=(-1, 1))
    ax.set_aspect('equal')
    plt.xlabel('пространственная координата $x$')
    plt.ylabel('возмущение $\\xi$')
    line, = ax.plot([], [])
    plt.margins(0.5)
    
    xs = np.linspace(0, 3 * np.pi, num=1000)

    def animate(t):
        ys = wave_func(xs, t)
        line.set_data(xs, ys)
        return line

    anim = FuncAnimation(fig, animate,
                        frames=np.linspace(0, 3 * np.pi, 30 * 3),
                        interval=1000/30)

    # show video
    video = anim.to_html5_video()
    html = display.HTML(video)
    display.display(html)
    plt.close()

wave_shape = lambda x: x * np.exp(-x ** 2)
wave_func = lambda x, t: wave_shape(x - t)
plot(wave_func)

Для гармонической1 волны с заданной частотой колебания \(\omega\), распространяющейся вдоль оси \(x\), аналитическое решение имеет простой вид: \[ \xi(x, t) = A \sin (\omega t - k x), \] где \(k := \omega / v\) — волновое число.

  • 1 т.е. синосуидальной

    • Более обще, волновыми называют уравнения, имеющие решения схожие с (TODO), т.е. \[ \xi = f(\bf r + \bf v t), \] где \(f\) — форма волны, \(v\) — вектор скорости. Решения такого вида могут быть получены из уравнения Шрёдингера, поэтому оно тоже называется волновым.

    \[\Psi(x, t) = A e^{i (k x - \omega t)} \tag{2.1}\]

    Волна де Бройля

    В специальной теории относительности связь полной энергии частицы \(E\) с её массой покоя \(m_0\) определяется выражением: \[E^2 = m_0^2 c^4 + p^2 c^2,\] где \(c\) — скорость света, \(p\) — импульс частицы.

    Связь энергии фотона с постоянной Планка \(h\) и длиной волны \(\lambda\): \[E = h \nu = \dfrac{h c}{\lambda},\] воспользуемся этим, приняв также массу покоя фотона \(m_0 = 0\), и подставив в [TODO] получим сначала \[\dfrac{h^2 c^2}{\lambda^2} = p^2 c^2\] и затем выражение, связывающее импульс фотона с длиной его волны: \[\lambda = \dfrac{h}{p}.\]

    Не смотря на то, что [TODO] выведено для фотона, Луи де Бройль (1924) использовал эту формулу для объяснения фотоэффекта, предположив, что волновыми свойствами обладают и другие частицы, и получил связь длины волны электрона с его импульсом: \[\lambda_e = \dfrac{h}{p},\] что также подтверждается серией экспериментов Дэвиссона и Джермера (1923-27). Открытие волновых свойст материи — волн де Бройля — позволило объяснить противоречия, возникающие при построении полуклассической Боровской модели атома (1913), стало возможно описание элементарных частиц с позиции современной квантовой механики — в терминах волн и вероятностей.
